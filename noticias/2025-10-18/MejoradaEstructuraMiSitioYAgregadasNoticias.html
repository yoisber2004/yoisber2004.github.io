Mejorada la estructura de mi sitio web y agregada una nueva sección de noticias

Hace unos días fue realizado un rediseño completo de mi sitio web para que estubiera apto para las publicaciones sobre mis juegos y recursos para los desarrolladores. De echo, la sección de recursos para los desarrolladores ya había sido agregada, para ir publicando próximamente algunos módulos python desarrollados por mi.

Pero ahora han llegado cambios en la estructura, para que el sitio sea más compacto y fácil de navegar. Además, se ha agregado una nueva sección de noticias, donde estaré publicando periódicamente novedades sobre los juegos que esté desarrollando y otros proyectos relacionados, como los recursos para los desarrolladores que les había mencionado previamente. Con esta noticia damos inicio a esta nueva sección, que ha llegado para quedarse.

Estas son las novedades que han llegado:

  • Rediseño completo del sitio web para una mejor experiencia de usuario.
  • Agregada la sección de noticias para mantener a los visitantes informados sobre las novedades y actualizaciones.
  • Mejorada la navegación con un menú más compacto y accesible, Pudiendo expandirlo y contraerlo fácilmente en todas las páginas.
  • Preparación para futuras publicaciones de módulos python en la sección de recursos para los desarrolladores.

¡sigan atentos, que se bienen más novedades!
