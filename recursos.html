Recursos para los desarrolladores

Aquí encontrarás algunos recursos útiles para desarrolladores que he desarrollado, especialmente para facilitar el desarrollo de juegos accesibles en python con pygame. Para poder aprovechar estos recursos, es importante tener instaladas las siguientes herramientas de las que depende:

  • Python: Es el lenguaje de programación en el que están desarrollados estos juegos, y es importante aprenderlo si se desea contribuir con el desarrollo del mismo. Nota: En el caso de live history, hay un 80% de probabilidad de que sea desarrollado en nvgt, por lo que también es importante aprender a programar en este motor de juegos.
  • Pygame: Es el motor de juegos más conocido de python, el cual envuelve la librería sdl de c++. Es sobre este motor de juegos, pygame, que están desarrollados todos los módulos python que verán en esta página.
  • Accessible output 3/sral: Son librerías responsables de la accesibilidad con el lector de pantalla, y son la clave para hacer software completamente accesible en python.
  • Pyperclip: Módulo que aprovecha la API del sistema bajo el que se ejecuta para copiar y pegar contenido del portapapeles, principalmente necesario en los módulos historia y audio interfaz publicados aquí.
